@RTK.md

Préfère `rg` à `grep` pour les recherches dans le code.  
Préfère `fd` à `find` pour les recherches de fichiers.  
Préfère `bat` à `cat` pour afficher le contenu de fichiers.  
Utilise `jq` pour parser/filtrer du JSON en ligne de commande.  
Utilise `yq` pour parser/filtrer du YAML en ligne de commande.  

Ne jamais utiliser de tiret cadratin (—) dans les textes rédigés.
